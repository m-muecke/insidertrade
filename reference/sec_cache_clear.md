# Clear the cache

Deletes all cached ZIP files downloaded by
[`sec_form345()`](https://m-muecke.github.io/insidertrade/reference/sec_form345.md).

## Usage

``` r
sec_cache_clear()
```

## Value

`NULL` invisibly.
